本文件夹用于存放以下资料：
1，标准例程-IDF版。这个文件夹包含所有的IDF版本实例源码（不能在中文路径下打开工程）。
2，标准例程-MicroPython版。这个文件夹包含所有的MicroPyhton版本实例源码。
3，标准例程-Arduino版。这个文件夹包含所有的Arduino版本实例源码。
4，扩展例程-IDF版。这个文件夹里面包含正点原子最新发布的例程，包括：包含FreeRTOS、WiFi、AI例程。
5，扩展例程-MicroPython版。这个文件夹里面包含正点原子最新发布的例程，包括：包含WiFi&Blue、AI例程。
6，扩展例程-Arduino版。这个文件夹里面包含正点原子最新发布的例程，包括：WiFi&Blue例程。

为了省空间和提高刻录成功率，我们把该目录下文件夹都压缩成了.rar格式，需大家拷贝到电脑后自行解压一下。
